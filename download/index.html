Unsigned builds - what to expect

Pre-release Miccy packages are distributed without an Apple, Microsoft, or Linux vendor signature. Your operating system may block or warn on first launch. That is normal until we publish signed installers.

  • macOS: If Gatekeeper says the app “can’t be opened,” open System Settings → Privacy & Security and choose Open Anyway, or right-click the app → Open → confirm. You may need to remove the quarantine flag in Terminal (xattr -dr com.apple.quarantine /path/to/Miccy.app) if the app was downloaded from the web.
  • Windows: SmartScreen may show “Windows protected your PC.” Click More info → Run anyway if you trust this download source.
  • Linux: For AppImage, run chmod +x Miccy*.AppImage then execute from a terminal or file manager. For archives, extract and run the binary from a terminal so you can see any missing dependency messages.
